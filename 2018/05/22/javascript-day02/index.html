javascript day02

学习时间：3h

说明：今天主要还是学习js基础，包括数据类型等。

defer属性：

延迟脚本：脚本会被延迟到整夜页面都解析完毕后再运行，但但只适用于外嵌式 格式：defer=’defer’。

async属性：

与defer类似，只适用于外部脚本文件，并告诉浏览器立即下载文件，但不能保证先后顺序

如果在函数中适用var定义一个变量，那么在函数退出时变量就会被销毁；如果省略var，那么就会定义一个全局变量。

可以用var定义多个变量，之间用逗号隔开就好。

数据类型： Undefined Null Boolean Number

  • undefined和null都是只有一个值的数据类型。
  • alert未定义的变量会出错，但是typeof未定义的变量确会显示undefined；
  • null值表示一个空对象指针。
  • undefined派生自null值，所以 alert(null == undefined)； //true

Boolean类型：

只有两个字面值：true 和false（跟‘1’和‘0’没有关系，true不一定等于1，fasle也不一定等于0）

Number类型：

科学计数法：

var floatNum=3.125e7 // 等于31250000

永远不要测试某个特定的浮点数值

例如：if（a+b == 0.3）

如果这两个数是0.05和0.25或者都是0.15那么不会有问题，但如果是0.1和0.2，那么测试将无法通过

NaN（not a number）：

isNaN() 函数：用来判断参数是否“不是数值”。

数值转换：

Number（） parseInt（） parseFloat（）

第一个函数可以用于任何数据类型 另外两个函数则专门用于把字符串转换成数值。

Number()函数的转换规则如下。

如果是Boolean值，true和false 将分别被转换为1和0。如果是数字值，只是简单的传人和返回。如果是nul1值，返回0。

如果是undefined,返回NaN。

如果是字符串，遵循下列规则:

  • 如果字符串中只包含数字(包括前面带正号或负号的情况)，则将其转换为十进制数值，即” 1”会变成1, “123”会变成123，而”011”会变成11(注意: 前导的零被忽略了);

  • 如果字符串中包含有效的浮点格式，如” 1.1”，则将其转换为对应的浮点数值(同样，也会忽略前导零);

  • 如果字符串中包含有效的十六进制格式，例如”0xf”,则将其转换为相同大小的十进制整数值;
  • 如果字符串是空的(不包含任何字符),则将其转换为0;
  • 如果字符串中包含除上述格式之外的字符，则将其转换为NaN。
  • 如果是对象，则调用对象的valueOf () 方法，然后依照前面的规则转换返回的值。如果转换的结果是NaN,则调用对象的toString () 方法，然后再次依照前面的规则转换返回的字符串值。

parseInt（）最常用。

String类型：

要把值转变为string类型：toString()方法。可以不带参数，默认情况下是以十进制返回数值的字符串表示，toString(2):转换成二进制表示数值的字符串。

Object类型：

创建对象：

var o = new Object()

valueof():返回对象的字符串、数值或者布尔值表示，通常与toString()方法的返回值相同

操作符：

++age –age //注意++age和 age++的区别、这个操作符只能用于变量，不能用于常量。

这些操作符还可以运用在字符串、布尔值、浮点数值和对象。

位操作符：
非操作符：~           与操作符：&          或操作符：|
布尔操作符：
逻辑非：！               逻辑与：&&           逻辑或：||
运算操作符：* 、/、+、-、%

全等和不全等：==、===

“55” == 55 // true

“55” === 55 //false

三目运算符：格式: 条件表达式 ? 语句1 : 语句2;

函数：

格式：

function functionName(arg0,arg1….){}

ECMAScript函数没有重载。

JavaScript变量 作用域：

变量的复制：

var num1 = 10;

var num2 = num1;

这个例子中，两个变量可以参与任何操作而不相互影响；

var obj1 = new Object();
var obj2 = obj1;
obj1.name = “Nicholas”;
; //Nicholas
obj2.name = “wang yu lei”;
; // wang yu lei
当变量间复制引用类型的值时，就有不同之处。这个副本实际是一个指针，指向存储在堆中的一个对象。复制结束后两个变量实际上将引用同一个对象。因此，改变其中一个变量就会影响另一个变量。
