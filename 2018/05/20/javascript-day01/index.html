javascript学习（day 1）

学习时间：2h

初识js：

如何使用变量？

答：var a = xxx 这里的 = 是赋值运算符 规范:习惯将 = 的两侧 各加上一个 空格。

什么是关键字？

答：被JavaScript语言赋予了特殊含义的单词，在开发工具中，关键字会显示特殊的颜色。

什么是标识符：

答：编程时起的名字，例如 var num 这里面的num就是一个标识符。

标识符命名规则(必须遵守)

只能由26个英文字母的大小写、10个阿拉伯数字0~9、下划线_、美元符号$组成
不能以数字开头
严格区分大小写，比如test和Test是2个不同的标识符
不可以使用关键字、保留字作为标识符
JS底层保存标识符时实际上是采用的Unicode编码，所以理论上讲，所有的utf-8中含有的内容都可以作为标识符
标识符命名规范(建议遵守)
变量的名称要有意义(有利于提高阅读性)
变量的命名遵守驼峰命名法，首字母小写,第二个单词的首字母大 例如：useName，myFunction

在页面中显示内容：

document.write(“hello
world2”);

在控制台中显示内容：

console.log(“hello world3”); console.error(“错误信息”); console.warn(“警告信息”);

innerHTML与value的区别：
innerHTML是标签中的html，value则是一个值。
