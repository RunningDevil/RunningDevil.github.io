学习时间：2h

学习内容：主要是js引用类型

ECMAScript引用类型

引用类型是一种数据结构，用于将数据和功能组织在一起，它也常被称为类（class），引用类型有时也被称为对象定义，因为它们描述的是一类对象所具有的属性和方法。

Object类型

创建方法有两种：

new操作符后跟Object构造

var person = new Object();

//如果不给构造函数传递参数可以不加括号 var person = new Object;

person.name = ‘bai’;

person.age = 29;

使用对象字面量

var person = {

name : ‘bai’,

age : 29,

5 : true};

设置对象：

有两种访问对象属性的方法，可以用点表示法或中括号表示法获取、设置或更新对象的属性：

person.白或person[‘白’]

删除对象：
格式：delete person.白

elete只能删除对象下的数据，其他5种基础类型的值是删除不掉的

对象嵌套：

var student = {

name : {

a : 1,

b : 2

},

sex : 1,

age : 26

}

取值只能一层一层取，如student.name.chinese，而不能跨过name，直接用student.chinese

Array对象：

命名格式：

1、var mycars=new Array(“Saab”,”Volvo”,”BMW”)

2、var mycars=new Array(3)
mycars[0]=”Saab”
mycars[1]=”Volvo”
mycars[2]=”BMW”
括号里面的‘3’可以省略

String对象：

使用String对象的长度属性来计算字符串的长度：

var txt=”Hello world!”
document.write(txt.length)

字符串对象的toUpperCase()方法将字符串转换为大写：

var txt=”Hello world!”
document.write(txt.toUpperCase())

日期对象：

使用 Date() 方法获得当日的日期

如何使用 setFullYear() 设置具体的日期。

详情见w3cschool：http://www.w3school.com.cn/js/js_obj_date.asp

Boolean（逻辑）对象：

格式：var myBoolean=new Boolean()
下面的所有的代码行均会创建初始值为 false 的 Boolean 对象。
var myBoolean=new Boolean();
var myBoolean=new Boolean(0);
var myBoolean=new Boolean(null);
var myBoolean=new Boolean(“”);
var myBoolean=new Boolean(false);
var myBoolean=new Boolean(NaN);

各种方法：

  • 数组检测：Array.isArray（）

  • tostring和valueof的区别：

  • valueOf()：返回最适合该对象类型的原始值；

  • toString(): 将该对象的原始值以字符串形式返回。

栈方法：

  • push（）：接收任意参数，逐个添加到数组末尾，然后返回数组长度

  • pop（）：从数组末尾移除最后一项，减小数组的length值，然后返回移除的项

队列方法：shift（）和unshift（）方法。

重排序方法：reverse（）和sort（）；前者反转数组顺序，后者按升序排列，但是比较的是字符串。

后者要实现对数值大小的排序，则需要使用一个排序函数

contat() splice()方法狠强大。详情见http://www.w3school.com.cn/jsref/jsref_splice.asp

javascript day02

学习时间：3h

说明：今天主要还是学习js基础，包括数据类型等。

defer属性：

延迟脚本：脚本会被延迟到整夜页面都解析完毕后再运行，但但只适用于外嵌式 格式：defer=’defer’。

async属性：

与defer类似，只适用于外部脚本文件，并告诉浏览器立即下载文件，但不能保证先后顺序

如果在函数中适用var定义一个变量，那么在函数退出时变量就会被销毁；如果省略var，那么就会定义一个全局变量。

可以用var定义多个变量，之间用逗号隔开就好。

数据类型： Undefined Null Boolean Number

  • undefined和null都是只有一个值的数据类型。
  • alert未定义的变量会出错，但是typeof未定义的变量确会显示undefined；
  • null值表示一个空对象指针。
  • undefined派生自null值，所以 alert(null == undefined)； //true

Boolean类型：

只有两个字面值：true 和false（跟‘1’和‘0’没有关系，true不一定等于1，fasle也不一定等于0）

Number类型：

科学计数法：

var floatNum=3.125e7 // 等于31250000

永远不要测试某个特定的浮点数值

例如：if（a+b == 0.3）

如果这两个数是0.05和0.25或者都是0.15那么不会有问题，但如果是0.1和0.2，那么测试将无法通过

NaN（not a number）：

isNaN() 函数：用来判断参数是否“不是数值”。

数值转换：

Number（） parseInt（） parseFloat（）

第一个函数可以用于任何数据类型 另外两个函数则专门用于把字符串转换成数值。

Number()函数的转换规则如下。

如果是Boolean值，true和false 将分别被转换为1和0。如果是数字值，只是简单的传人和返回。如果是nul1值，返回0。

如果是undefined,返回NaN。

如果是字符串，遵循下列规则:

  • 如果字符串中只包含数字(包括前面带正号或负号的情况)，则将其转换为十进制数值，即” 1”会变成1, “123”会变成123，而”011”会变成11(注意: 前导的零被忽略了);

  • 如果字符串中包含有效的浮点格式，如” 1.1”，则将其转换为对应的浮点数值(同样，也会忽略前导零);

  • 如果字符串中包含有效的十六进制格式，例如”0xf”,则将其转换为相同大小的十进制整数值;
  • 如果字符串是空的(不包含任何字符),则将其转换为0;
  • 如果字符串中包含除上述格式之外的字符，则将其转换为NaN。
  • 如果是对象，则调用对象的valueOf () 方法，然后依照前面的规则转换返回的值。如果转换的结果是NaN,则调用对象的toString () 方法，然后再次依照前面的规则转换返回的字符串值。

parseInt（）最常用。

String类型：

要把值转变为string类型：toString()方法。可以不带参数，默认情况下是以十进制返回数值的字符串表示，toString(2):转换成二进制表示数值的字符串。

Object类型：

创建对象：

var o = new Object()

valueof():返回对象的字符串、数值或者布尔值表示，通常与toString()方法的返回值相同

操作符：

++age –age //注意++age和 age++的区别、这个操作符只能用于变量，不能用于常量。

这些操作符还可以运用在字符串、布尔值、浮点数值和对象。

位操作符：
非操作符：~           与操作符：&          或操作符：|
布尔操作符：
逻辑非：！               逻辑与：&&           逻辑或：||
运算操作符：* 、/、+、-、%

全等和不全等：==、===

“55” == 55 // true

“55” === 55 //false

三目运算符：格式: 条件表达式 ? 语句1 : 语句2;

函数：

格式：

function functionName(arg0,arg1….){}

ECMAScript函数没有重载。

JavaScript变量 作用域：

变量的复制：

var num1 = 10;

var num2 = num1;

这个例子中，两个变量可以参与任何操作而不相互影响；

var obj1 = new Object();
var obj2 = obj1;
obj1.name = “Nicholas”;
; //Nicholas
obj2.name = “wang yu lei”;
; // wang yu lei
当变量间复制引用类型的值时，就有不同之处。这个副本实际是一个指针，指向存储在堆中的一个对象。复制结束后两个变量实际上将引用同一个对象。因此，改变其中一个变量就会影响另一个变量。

javascript学习（day 1）

学习时间：2h

初识js：

如何使用变量？

答：var a = xxx 这里的 = 是赋值运算符 规范:习惯将 = 的两侧 各加上一个 空格。

什么是关键字？

答：被JavaScript语言赋予了特殊含义的单词，在开发工具中，关键字会显示特殊的颜色。

什么是标识符：

答：编程时起的名字，例如 var num 这里面的num就是一个标识符。

标识符命名规则(必须遵守)

只能由26个英文字母的大小写、10个阿拉伯数字0~9、下划线_、美元符号$组成
不能以数字开头
严格区分大小写，比如test和Test是2个不同的标识符
不可以使用关键字、保留字作为标识符
JS底层保存标识符时实际上是采用的Unicode编码，所以理论上讲，所有的utf-8中含有的内容都可以作为标识符
标识符命名规范(建议遵守)
变量的名称要有意义(有利于提高阅读性)
变量的命名遵守驼峰命名法，首字母小写,第二个单词的首字母大 例如：useName，myFunction

在页面中显示内容：

document.write(“hello
world2”);

在控制台中显示内容：

console.log(“hello world3”); console.error(“错误信息”); console.warn(“警告信息”);

innerHTML与value的区别：
innerHTML是标签中的html，value则是一个值。
